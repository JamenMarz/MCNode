getNameHistory

Gets all name changes for a given player's profile ID.

getNameHistory(profileId: String): Promise<Array>
Parameters
profileId (String) Required. An existing profile ID (does not work with user ID).
Returns
Promise<Array>: that resolves with [{name, changedToAt}] .

getUserInfo

Gets a user's private Mojang account information.

getUserInfo(accessToken: String): Promise<MojangUser>
Parameters
accessToken (String) A valid access token for the user's account.
Returns
Promise<MojangUser>: that resolves if access token is valid.

getUuidAt

Gets a username at a given timestamp.

getUuidAt(name: String, date: Number?, agent: String?): Promise<Object>
Parameters
name (String) Current profile name of the user
date (Number?) UNIX timestamp to check the username at
agent (String? = 'minecraft') Game agent to check against
Returns
Promise<Object>: that resolves with {id, name, legacy, demo} .
Example
const {id, name, legacy, demo} = await getNameAt('Notch')

getProfile

A special service that returns a profile's name, skin, cape, and maybe more.

Rate limit: You can request the same profile once per minute.

getProfile(profileId: String): Promise<Object>
Parameters
profileId (String) required. UUID of a player
Returns
Promise<Object>: Promise which resolves to {id, name, properties}

authenticate

Authenticates a user with their Mojang credentials.

authenticate(username: String, password: String, clientToken: String, agent: Object): Promise<Object>
Parameters
username (String) required. Valid username of a Mojang account
password (String) required. Password for the given account
clientToken (String) optional. Client token for the request
agent (Object) optional. If valid, adds selectedProfile to response
Name Description
agent.name String The name of the agent. Can be "Minecraft" or "Scrolls"
agent.version Number Version number of the agent. Use 1 as a default value
Returns
Promise<Object>: Promise which resolves to a session {clientToken, accessToken, selectedProfile, user}

invalidate

Invalidate an obtained access token.

invalidate(accessToken: String, clientToken: String): Promise
Parameters
accessToken (String) required. Access token to be invalidated
clientToken (String) optional. Must match the one used to obtain the access token
Returns
Promise: always resolves, regardless if tokens were valid

refresh

Refreshes a given access token.

refresh(accessToken: String, clientToken: String, selectedProfile: Object): Promise<Object>
Parameters
accessToken (String) required. Valid access token
clientToken (String) required. Must match the one used to obtain the access token
selectedProfile (Object) optional. Only use with access tokens that were not assigned a game profile
Name Description
selectedProfile.id String Player id
selectedProfile.name String Player name
Returns
Promise<Object>: Promise which resolves to a session {clientToken, accessToken, selectedProfile, user}
