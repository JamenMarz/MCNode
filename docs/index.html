getNameHistory

Gets all name changes for a given player's profile ID.

getNameHistory(profileId: String): Promise<Array>
Parameters
profileId (String) profile UUID (does not work with user ID)
Returns
Promise<Array>: resolves with [{name, changedToAt}] .

getProfileAt

Gets an abbreviated game profile from a given timestamp.

getProfileAt(name: String, date: Number?, agent: String?): Promise<Object>
Parameters
name (String) current profile name (IGN) of the user
date (Number?) UNIX timestamp to check the username at
agent (String? = 'minecraft') game agent to check against
Returns
Promise<Object>: resolves with {id, name, legacy, demo}
Example
const {id, name, legacy, demo} = await getProfileAt('Notch')

setSkin

Sets a logged-in user's in-game skin to an online image.

setSkin(accessToken: String, profileId: String, skinUrl: String, isSlim: Boolean?): Promise
Parameters
accessToken (String) valid access token for the user's account
profileId (String) which profile to update the skin of (Minecraft profile UUID)
skinUrl (String) Internet-accessible URL of the image file
isSlim (Boolean? = false) if true, the slim model will be used with the skin
Returns
Promise: resolves if access token, profile ID, and skin URL are valid

authenticate

Authenticates a user with their Mojang credentials.

authenticate(username: String, password: String, clientToken: String?, agent: Object?): Promise<Object>
Parameters
username (String) username (email) of a Mojang account
password (String) password for the given account
clientToken (String?) server will generate a client token if empty
agent (Object?) if valid, adds selectedProfile to response
Name Description
agent.name String name of the agent ('Minecraft' or 'Scrolls')
agent.version Number version number of the agent (use 1 )
Returns
Promise<Object>: resolves to a session {clientToken, accessToken, selectedProfile, user}

invalidate

Invalidate an access token.

invalidate(accessToken: String, clientToken: String?): Promise
Parameters
accessToken (String) a session access token
clientToken (String?) must match the one used to obtain the access token
Returns
Promise: always resolves, regardless if tokens were valid

refresh

Refreshes a given access token. May work on invalid access tokens.

refresh(accessToken: String, clientToken: String, selectedProfile: Object?): Promise<Object>
Parameters
accessToken (String) a session access token
clientToken (String) must match the one used to obtain the access token
selectedProfile (Object?) only use with access tokens that were not assigned a game profile
Name Description
selectedProfile.id String profile ID
selectedProfile.name String profile name (IGN)
Returns
Promise<Object>: resolves to a new session {clientToken, accessToken, selectedProfile, user}

getProfile

A special service that returns a profile's name, skin URL, and cape URL.

This endpoint is used by the official launcher to load the current skin.

Rate limit: You can request the same profile once per minute.

getProfile(profileId: String): Promise<Object>
Parameters
profileId (String) profile UUID of a player
Returns
Promise<Object>: resolves to {id, name, timestamp, skin, cape, isSlim}
